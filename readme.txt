Налаштунки
1) Встановити всі залежності з package.json в папках client, server і в папку проекту.
2) В папці server лежить файлик .env.example як приклад потрібного для .env
3) В папці client лежить файлик .env.example як приклад потрібного для .env. Містить домен server api. (Для запуску локально значення хоста залишаєм)

Локально запустити можна командою npm run dev з кореня проекту.

В проекті реалізовано
1) Серверну частину з усіма необхідними ендпоінтами.
2) Клієнтська частина з усім функціоналом.
3) Нових героїв можна створити або видалити.
4) Зображення на вибір можна видаляти або додавати.
5) Поля тексту можна або модифікувати або робити пустими.
6) Реалізований слайдер.
7) Реалізована пагінація.

Верстка демонстраційна(не responsible/adaptive), функціональна частина в нормі.
За потреби можу розписати тести на всю апку.

Є маршрутизація react-router-dom (1 сторінка)
Є обробка помилок, на стороні сервера і клієнта.
Є написаний тест для сервера (1 роут + звірка через модель)

Стек:
Клієнт: React, Redux Toolkit, SASS.
Сервер: Nodejs, Express, Multer, MongoDB/Mongoose.
Тест: Jest, supertest
